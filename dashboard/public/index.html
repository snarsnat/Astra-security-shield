About ASTRA

Behavioral bot detection and human verification system.

ASTRA Shield is a security framework that analyzes user behavior patterns to distinguish humans from bots. It uses a 5-tier friction model where 95% of legitimate users experience zero friction, and only suspicious traffic faces additional verification.

This dashboard reads data from apps registered via the astra add CLI command. Each app's API routes send verification data to this dashboard for aggregation and display.
